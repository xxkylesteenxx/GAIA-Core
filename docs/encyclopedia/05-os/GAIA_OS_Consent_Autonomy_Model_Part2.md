## §3 — Always-On Feature Consent Protocols

### §3.1 — Geo-Location Consent Framework

| Consent Category | Scope |
|---|---|
| **Precise Location** | GPS coordinates · real-time tracking (emergency) · location history · movement pattern analysis |
| **Approximate Location** | City/region level · time zone detection · regional content/language · area-based recommendations |
| **Location Context** | Home/work identification · frequent pattern recognition · location-based automation · contextual service activation |
| **Location Sharing** | Family/friend sharing · emergency contact access · service provider sharing · third-party app access |
| **Data Retention** | Session-only · 24–48 hours · weeks/months (patterns) · years (analytics) |

|---|---|

| **Travel Mode** | Enhanced location services · travel recommendations · emergency contact protocols · international privacy law compliance |
