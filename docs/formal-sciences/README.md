## Systems Architecture Cross-Reference (Volume C §6)

### Closed-Loop Canonical Architecture

| Layer | Function |
|---|---|
| 1. Measurement | Sensor and observational input |
| 2. Inference | Estimation, filtering, anomaly detection |
| 3. Model | System state and forecast dynamics |
| 4. Constraint | Legal, ethical, ecological, operational limits |
| 5. Decision | Recommendation or control-law selection |
| 6. Actuation | Execution only under proper authorization |
| 7. Audit | Proof/evidence of why intervention occurred |
| 8. Learning | Recalibration after outcome observation |

| Scale | Formal Concern |
