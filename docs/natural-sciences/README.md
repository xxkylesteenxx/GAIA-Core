## Series Purpose

The Natural Sciences series provides the **empirical reality constraints** for GAIA. Where the Formal Sciences series defines what can be expressed and computed, the Natural Sciences series defines what is physically, chemically, biologically, and geologically real. GAIA’s models must be grounded in both.

### Relationship to Formal Sciences

| Formal Sciences | Natural Sciences |
|---|---|
| Define valid inference | Define physical law |
| Define structure and computation | Define material behavior |
| Define feedback and control | Define what can be sensed and actuated |
| Define uncertainty calculus | Define the sources of measurement uncertainty |
